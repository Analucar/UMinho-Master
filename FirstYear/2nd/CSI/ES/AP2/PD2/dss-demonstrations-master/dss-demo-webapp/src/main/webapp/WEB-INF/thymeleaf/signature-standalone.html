This demo is a standalone application which uses JavaFX.

The application connects directly to the CA's infrastructure to retrieve information such as CRL, OCSP, certificates from AIA ...

All DSS business logic is embedded inside this application (CAdES, PAdES, XAdES, ASiC). This application doesn't require a DSS server.
